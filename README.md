# 🎁 Gifting Cards Website

A modern and user-friendly gifting cards website designed to provide a smooth and elegant user experience. This platform allows users to explore, select, and purchase beautifully designed gift cards for different occasions.

## 🚀 Features
- 🎨 Modern and responsive design
- 🛒 Easy navigation and product browsing
- 💳 Secure and smooth user experience
- 📱 Mobile-friendly layout

## 🛠️ Technologies Used
- HTML5
- CSS3
- Bootstrap
- JavaScript

## 📂 Project Setup
1. Clone the repository
2. Open the project folder
3. Run index.html in your browser
